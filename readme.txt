AI 교육제안 메일 자동 생성기
=======================

스크린샷 또는 텍스트를 입력하면 AI가 분석하여 맞춤 교육과정을 매칭하고 제안 메일을 자동 생성하는 도구입니다.

주요 기능:
- 듀얼 입력: 좌우 2열 레이아웃 — 왼쪽 텍스트 붙여넣기(오렌지) + 오른쪽 스크린샷 업로드(파란)
- 스크린샷 이미지 OCR 텍스트 추출 (Claude Sonnet 4.5 Vision)
- 텍스트 입력: 붙여넣기 자동 감지(300ms 디바운스, 10자 이상) → 큐 추가 → AI 분석
- 기관명, 교육대상, 목적 자동 파악
- 나눔경영컨설팅 교육과정 100여개 중 최적 2~4개 자동 매칭
- Gmail 붙여넣기용 인라인 HTML 메일 자동 생성
- 메일주소/제목/본문 개별 복사 기능
- 다중 파일 큐 처리 (이미지+텍스트 혼합 지원)
- 네이버 밴드용 OCR 원문 복사
- URL 파라미터 연동: ?text= 파라미터로 외부(Edu-CRM)에서 텍스트 전달 → 자동 분석
- 반응형: 768px 이하에서 세로 스택

기술:
- 프론트엔드: HTML/CSS/JS (단일 파일)
- AI: Anthropic Claude Sonnet 4.5 API (Vision + Text)
- 배포: GitHub Pages
- 연동: Edu-CRM 제안메일 버튼에서 URL 파라미터로 호출

제작: 김종혁 (나눔경영컨설팅)
