AI 교육제안 메일 자동 생성기
=======================

스크린샷을 드래그 앤 드롭하면 OCR 분석 후 맞춤 교육과정을 매칭하여 제안 메일을 자동 생성하는 AI 도구입니다.

주요 기능:
- 스크린샷 이미지 OCR 텍스트 추출
- Claude Sonnet 4.5 API Vision으로 기관명, 교육대상, 목적 자동 파악
- 나눔경영컨설팅 교육과정 100여개 중 최적 2~4개 자동 매칭
- Gmail 붙여넣기용 인라인 HTML 메일 자동 생성
- 메일주소/제목/본문 개별 복사 기능
- 다중 파일 큐 처리 (여러 이미지 동시 업로드)
- 네이버 밴드용 OCR 원문 복사

기술:
- 프론트엔드: HTML/CSS/JS (단일 파일)
- AI: Anthropic Claude Sonnet 4.5 API (Vision)
- 배포: GitHub Pages

제작: 김종혁 (나눔경영컨설팅)
